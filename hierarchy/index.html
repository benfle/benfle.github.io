Hierarchy

#

Hierarchy is a modularity technique used in many complex systems.

#

The coupling of the system is reduced by distinguishing different levels and only allowing a level to interact with the level directly above and below.

#

A hierarchical description of a system often results from adopting a top-down approach.

#

In "The Architecture of Complexity", Herbert Simon uses a metaphor about watchmakers to show the difference in evolution between a system that is hierarchically organized and one that is not.

Causal Loop Diagrams

#

Causal loop diagrams show the causal relationships between the variables of a system.

#

A causal loop diagram is a mental tool part of systems thinking.

#

The diagram helps us identifying the positive and negative feedback loops in order to understand the Stability of the system. A positive feedback loop will cause an explosion. A negative feedback loop will cause stability.

#

Both explosion and stability is desirable in some context and undesirable in others.

#

Nicky Case made an online tool, LOOPY, to make interactive causal loop diagrams.

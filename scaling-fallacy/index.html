Scaling Fallacy

#

The fallacy to think that large systems are like small systems, just bigger.

#

The problem is that not all the variables of a system change at the same rate.

#

We can see it with a simple cube. When we double the distance, the surface is four times bigger and the volume eight times bigger.

#

#

The term comes from Weinberg's Quality Software Management.

#

In complex systems, we can build causal loop diagrams to identify the relationships between the variables and how they will scale.
